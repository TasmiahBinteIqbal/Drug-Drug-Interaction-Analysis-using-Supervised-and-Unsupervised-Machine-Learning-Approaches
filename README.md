# Drug-Drug-Interaction-Analysis-using-Supervised-and-Unsupervised-Machine-Learning-Approaches

This project focuses on analyzing drug-drug interactions (DDIs) using both supervised and unsupervised machine learning techniques. The goal is to identify patterns in drug interactions and evaluate how different models perform in classification and clustering tasks.

Objectives
- Analyze a dataset of drug-drug interactions
- Apply supervised machine learning models for classification
- Use unsupervised learning (clustering) to discover hidden patterns
- Compare model performance using evaluation metrics
- Visualize results for better interpretability

Dataset
The dataset contains:
- Drug 1
- Drug 2
- Interaction Description

Methodology
 Preprocessing
- Text cleaning and lowercasing
- Lemmatization
- TF-IDF vectorization (with n-grams)
- Rule-based labeling of interaction types
 Supervised Models
- Logistic Regression
- Random Forest
- Support Vector Machine (SVM)
- K-Nearest Neighbors (KNN)
 Unsupervised Learning
- KMeans Clustering
 Visualization
- Confusion Matrix
- Feature Importance
- 2D/3D PCA Visualization

Results
- Supervised models achieved high accuracy, precision, recall, and F1-score
- Logistic Regression, Random Forest, and SVM performed the best
- KMeans clustering showed moderate alignment with predefined categories
- PCA visualization helped identify cluster separations and overlaps
